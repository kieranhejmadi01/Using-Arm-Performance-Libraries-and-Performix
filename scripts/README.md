# Vector Visualization Scripts

This directory contains Python scripts for visualizing 2D vector data generated by the proxy-src C++ project.

## Workflow

1. **Compile and run the C++ binary** to generate vector data
2. **Run the Python visualization script** to animate and display the data

## Prerequisites

Install required Python packages:
```bash
pip install numpy pandas matplotlib
```

## Step 1: Generate Data with C++ Binary

Build and run the main C++ program:
```bash
cd ..
make
./build/main
```

This will generate `vector_data.csv` in the project root directory.

## Step 2: Visualize the Data

### `visualize_vectors.py`

Animated visualization of 2D vector distributions using matplotlib.animation.

**Features:**
- Reads vector data from CSV files generated by the C++ binary
- Creates animated scatter plots showing points progressively
- Supports multiple data files (compares distributions side-by-side)
- Auto-scales axes based on actual data bounds
- Can display interactively or save animations to GIF files

**Usage:**

Display interactive animation:
```bash
python visualize_vectors.py
```

Visualize a specific data file:
```bash
python visualize_vectors.py vector_data.csv
```

Visualize multiple data files (side-by-side comparison):
```bash
python visualize_vectors.py file1.csv file2.csv file3.csv
```

Customize animation speed:
```bash
python visualize_vectors.py --fps 30
```

Save animation to GIF file:
```bash
python visualize_vectors.py --save animation.gif --fps 20
```

**Arguments:**
- `files`: CSV file(s) to visualize (default: vector_data.csv)
- `--fps N`: Animation frames per second (default: 20)
- `--save FILE`: Save animation to file instead of displaying interactively

## CSV Format

The script expects CSV files with the following format:
```
x,y,type
10.5,20.3,exponential
-5.2,8.1,exponential
...
```

Columns:
- `x`: X-coordinate of the vector
- `y`: Y-coordinate of the vector  
- `type`: Distribution type (e.g., "exponential", "gaussian", etc.)

## How the Animation Works

The visualization:
1. Loads all data points from the CSV file(s)
2. Auto-scales the plot axes to fit the data range with 10% padding
3. Animates by progressively adding points frame-by-frame
4. Displays a counter showing progress (e.g., "Points: 500/4194304")
5. Loops continuously

## Technical Details

- **Data Loading**: Uses `pandas.read_csv()` for efficient CSV parsing
- **Animation**: `matplotlib.animation.FuncAnimation` creates smooth progressive reveals
- **Multi-file Support**: Creates subplots (up to 2 columns) when multiple files are provided
- **Memory Efficient**: Loads all data upfront but only renders what's needed per frame
